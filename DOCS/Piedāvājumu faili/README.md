# DOCS/Piedāvājumu faili/ — original offer PDFs

Original full-resolution offer / portfolio PDFs (Kravu pārvadājumu piedāvājums).

**This folder is intentionally `.gitignore`-d.** One of the master PDFs is ~129 MB which
exceeds GitHub's 100 MB per-file limit, so the originals stay on the local workstation.

If the site needs a downloadable brochure, commit a **web-optimized** PDF (target <5 MB) to:

```
assets/web/brochures/<slug>.pdf
```
